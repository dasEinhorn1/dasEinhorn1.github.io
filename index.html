Projects

  • Fourier Art

    Fourier Art was the product of a 36 hour hackathon at Georgia Tech. It dynamically visualizes mp3 files fed into it via javascript.

    It is a sound visualizer which was constructed using p5.js, a js port of Processing which could handle the fast fourier transform, and paper.js, which allows for complex vector-based graphics using SVGs or the HTML canvas element.

    Currently, this project is still a work in progress. Last summer, I intend to added file uploading, and built out playback functionalities.

  • The Shape of History

    The Shape of History modernizes a data visualization method from the 19th century with current web capabilities.

    The website, which I completed in the spring of 2017, features a series different guided and unguided interactions with the piece. It aims to demonstrate the ways in which modern technologies can improve upon (or take away from) old data visualization techniques.

    The team which worked on this site is led by Dr. Lauren Klein, and featured two undergraduates, one being myself, and one grad student. My focus within the project dealt primarily with SVG manipulation.

    I presented this project at the 2017 Digital Humanities Conference at McGill University last summer.
