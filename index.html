Projects

  • Fourier Art

    Fourier Art was the product of a 36 hour hackathon at Georgia Tech . It is a sound visualizer which was constructed using p5.js, a js port of Processing which could handle the fast fourier transform, and paper.js, which allows for complex vector-based graphics using SVGs or the HTML canvas element.

    Currently, this project is still a work in progress. We intend to add file uploading, and to build out the comprehensiveness of our visualizer.
