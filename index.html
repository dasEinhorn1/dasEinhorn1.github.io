Projects

  • Fourier Art

    Fourier Art was the product of a 36 hour hackathon at Georgia Tech. It dynamically visualizes mp3 files fed into it via javascript.

    It is a sound visualizer which was constructed using p5.js, a js port of Processing which could handle the fast fourier transform, and paper.js, which allows for complex vector-based graphics using SVGs or the HTML canvas element.

    Currently, this project is still a work in progress. We intend to add file uploading, and to build out the comprehensiveness of our visualizer.

  • The Shape of History

    The Shape of History modernizes a data visualization method from the 19th century with current web capabilities.

    The website, which is currently under development, features a series different guided and unguided interactions with the piece. It aims to demonstrate the ways in which modern technologies can improve upon (or take away from) old data visualization techniques.

    The team working on this site is led by Dr. Lauren Klein, and features two undergraduates, one being myself, and one grad student. My focus within the project deals primarily with SVG manipulation, and I am currently working on a newer version of the site which will offer a more comprehensive and intuitive UI.
