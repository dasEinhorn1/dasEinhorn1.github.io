Documentation

Thank You for playing my game! It really means a lot to know people notice my work.

In case you feel stuck, here are some instructions for the game. Note: Unimplemented features will look like this

  1. Getting Started
    • You start off in the game with nothing. That is okay, you don't need anything yet.
    • This game is text-based, meaning you type commands to get around and interact with people and things.
    • To start off, you should know you have the following commands:
      • look- describes the room you are in
      • speak- opens a dialogue with things to say, or you can say what you want
      • search- searches a room, container, or body for items. Takes any items from containers
      • go- lists exits for each area, and allows you to choose exits by number
      • take- takes a specified item from a room. Like search, but for single items and more general use
      • inventory- shows items you have picked up during the game
      • help- prints out a help menu for quick reference
    • It seems like a lot to deal with, but it will feel natural asyou play longer.
    • Start off by typing 'look' to view your surroundings!
  2. Getting Around
    • In order to go from room to room, you should use 'go' to list your exit options.
    • Each exit option will have a number before it. To choose that exit, use 'go <number>' where <number> is the number for the exit.
    • Locked Areas
      • Some areas have certain requirements in order for you to enter or exit them.
      • Don't get stressed, just find another exit or find the thing to open it up.
      • If an exit is unavailable, a message will show up in this color.
  3. Speech
    • Use 'speak' to talk to characters in the game.
    • It works like 'go'
      • 'speak' lists options with corresponding numbers
      • 'speak <num>' picks an option
    • Your options will change over time too, as people's conversations go on.
  4. Scavenging
    • This game features items and collectibles which you can find and pick up. Sometimes, you need them to get to other areas in the game.
    • Items are either found exposed in rooms or stored in containers and on dead bodies.
    • Search:
      • To search an area for items and containers/bodies, or to take items from containers and bodies use 'search'
      • Alone, 'search' will find all items and containers in an area
      • 'search desk' would search a container called 'desk' in the current area
      • 'search 1' would search the 1st container in an area as listed by 'search'
    • Take:
      • To take items sitting in an area (i.e. not in a container) use "take"
      • You can use the item name: 'take iron bar'
      • Or the number: if 1) iron bar, then use 'take 1'
      • Either way works, and it is not case sensitive
  5. Inventory
    • To see cool stuff you've collected throughout the game, use 'inventory'
    • 'inventory item name' shows specific items in your inventory with the given item name rather than the whole thing.
    • 'inventory item #' shows a specific item by its item number position in your inventory.
